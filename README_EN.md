# Rift2D

[English](README_EN.md) | [中文](README.md)

Rift2D is a Unity 2D action RPG project in development.

This public repository is only a lightweight showcase for the project. The full Unity project, source code, project files, configuration pipeline, editor tools, and original assets are not included.

## Project Overview

Rift2D focuses on a compact gameplay loop:

- lobby preparation
- dungeon exploration
- combat and resource collection
- extraction and settlement
- long-term character and inventory growth

The project is built with Unity 2022.3 and C#, using a QFramework-style Model / System / Command / Controller / Event architecture.

## Current Features

- Unity 2D scene flow for boot, lobby, dungeon, and return-to-lobby gameplay
- Save slot flow for creating, loading, continuing, and deleting saves
- Inventory, equipment, resources, and settlement-oriented progression
- Dungeon exploration and extraction loop
- YooAsset-based resource loading pipeline
- Luban XML / CSV configuration workflow
- HybridCLR-oriented hot update boundary
- Minimal realtime multiplayer validation path for local and LAN testing
- Project-local editor and automation tooling
